# Code of Conduct

This is a small project, but the basics still matter.

Be respectful and assume good faith. Disagreement is fine — that's how projects
get better — but personal attacks, harassment, and dismissive or demeaning
comments are not. Treat people the way you'd want to be treated when you're the
new person asking a question.

If someone is making the project a worse place to be, open an issue (or reach out
privately) and I'll look into it. Reports will be handled discreetly.

This is adapted loosely from the
[Contributor Covenant](https://www.contributor-covenant.org), which is worth
reading if you want the longer version.
