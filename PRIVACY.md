# 개인정보 처리방침 (Privacy Policy) — 토큰깎이

최종 업데이트: 2026

## 요약
**토큰깎이는 어떠한 개인정보도 수집·저장·전송하지 않습니다.**

## 상세

### 데이터 수집
이 확장은 사용자의 어떠한 데이터도 수집하지 않습니다. 구체적으로:

- 사용자가 ChatGPT·Claude 입력창에 작성한 텍스트(프롬프트)는 **사용자의 브라우저 안에서만**
  처리됩니다.
- 입력 내용은 외부 서버로 전송되지 않으며, 어디에도 저장되지 않습니다.
- 분석 도구(analytics), 추적기(tracker), 광고 네트워크를 일절 사용하지 않습니다.
- 쿠키를 사용하지 않습니다.

### 권한
이 확장은 다음 사이트에서만 동작합니다:
- `chatgpt.com`
- `claude.ai`
- `claude.com`

위 사이트의 입력창 텍스트를 읽고 다듬는 것은 전적으로 사용자의 브라우저 내에서
이루어지며, 그 목적(한국어 표현을 간결하게 다듬어 토큰을 줄이는 기능) 외에는
어떠한 데이터 접근도 하지 않습니다.

### 외부 전송
없음. 이 확장은 네트워크 요청을 보내지 않습니다.

### 문의
문의 사항은 본 프로젝트의 GitHub 저장소 Issues를 이용해 주세요.
